Error Page

This is a standard error page that is shown when something goes wrong with the payment process.

Mostly this will feature a retry option where it relates to a timeout. Otherwise this can be used for any other errors.

Error code (optional)

The error code is optional but is useful to be able to tie back an error to a specific piece of code. For example, timeouts might show the same error message, however the code can be different so it's clear in a bug report where this timeout is ocurring.
