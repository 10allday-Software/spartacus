Overview

This styleguide is a living example of the UI used in Spartacus the single page app front-end for webpay.

Any examples here that represent pages are the exact same templates used in the front-end. In all cases styles are the same styles used in the front-end.

View controls

For each example there are some buttons that allow you to switch between various sizes. Plus you'll notice that each example is resizable via the drag handle on the bottom-right corner

Currently we support the following screen-size presets:

  • Trusted UI - This is the trusted UI window shown on FireFox OS devices. This is used within all the payment flows.
  • Android: The android size is a typical android device viewport size as an example as to what this will look like in Firefox for Android.
  • Full-screen mode uses the full-screen api to full screen the example element.
